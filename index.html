The latest stable releases of Chrome, Firefox, Opera and Opera Mobile are supported.

Install 4chan X

Chrome:
  1. Save the 4chan X file from the install button as 4chan_x.user.js.
  2. Drag and drop the file into the extensions page (chrome://chrome/extensions/).
Firefox:
  1. If you don't have Greasemonkey installed:
    1. Install Greasemonkey.
    2. Restart Firefox.
  2. Install 4chan X.
Opera:
  1. If you haven't set your userscripts folder:
    1. Go to Preferences (Ctrl + F12) → Advanced → Content → JavaScript Options → User JavaScript Folder.
    2. Chose what will be your userscripts folder.
    3. Enable User JavaScript on HTTPS and save your settings.
  2. Save the 4chan X file as 4chan_x.js into your userscripts folder.
Opera Mobile:
  1. If you haven't set your userscripts folder:
    1. Open User Prefs.
    2. Enable User JavaScript and User JavaScript on HTTPS.
    3. Set User JavaScript File to what will be your userscripts folder, /sdcard/js for example
    4. Save your settings.
  2. Save the 4chan X file as 4chan_x.js into your userscripts folder.

Before making one,

  1. Make sure your browser & 4chan X are up to date.
  2. Disable your other extensions & scripts.
  3. If your issue persists, report precise steps to reproduce and console errors, if any. Open your console with:
    • Ctrl + Shift + J on Chrome & Firefox
    • Ctrl + Shift + O on Opera.
Known issues:
  • Clearing your 4chan cookies will erase your settings!
Chrome:
  • 4chan X only runs on 4chan! Chrome will say that it runs on all domains due to its match patterns.
Firefox:
  • Scriptish is bugged, use Greasemonkey instead.
  • If you get an error stating localStorage is null, go to about:config and enable dom.storage.enabled.
  • If you get a Security error, make sure cookies are enabled.
  • I will probably not fix bugs. It, amazingly, isn't my job.
  • If you want to request a feature, ask Mayhem first. He may be a cunt, but he can code

DownThemAll for Firefox: download all images in a thread with this fast filter: /^http://images.4chan.org/.
*nix: Download all images in a thread with this shell function (add it to your .bashrc): 

dta() {
  wget -O - $1 |
  grep -Eo 'images.4chan.org/[^"]+' |
  uniq |
  xargs wget
}
